# Johnson & Johnson (johnson-and-johnson)

## APIs

### LifeScan API
The Johnson & Johnson LifeScan API provides programmatic access to blood glucose monitoring data and diabetes management tools. LifeScan develops the OneTouch brand of blood glucose meters, and the API enables developers to integrate glucose monitoring data into health applications and wellness platforms.
